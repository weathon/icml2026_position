Universal Aesthetic Alignment Narrows Artistic Expression

Wenqi Marshall Guo, Qingyun Qian, Khalad Hasan, Shan Du

Aesthetic alignment can overrule user intent. When users ask for disorder, low light, muddy color, awkward composition, grief, or visual decay, reward models and generators often drag the image back toward a polished average preference.

What This Page Resists

It resists the conference-project default: white page, centered cards, smooth gradients, and a calm promise that every model failure is really a feature. The design uses clashing color, dead spacing, default links, and pasted-together figure evidence because the paper is about expression that does not want to be prettified.

Functional Rules Kept Intact

  • Navigation remains ordinary links.
  • Important text keeps strong contrast.
  • Images have captions and alt text.
  • The noise toggle creates a plainer reading mode.
  • Errors from gallery loading are printed on the page.
