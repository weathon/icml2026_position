Universal Aesthetic Alignment Narrows Artistic Expression

Image generation and reward models default to a single, average notion of beauty. When users explicitly ask for anti-aesthetic, abstract, or emotionally negative imagery, these systems quietly sanitize, brighten, and smile back. We call this reversed alignment: instead of aligning the model to the user, the user gets aligned to the model.

Wenqi Marshall Guo, Qingyun Qian, Khalad Hasan, Shan Du
University of British Columbia · Weathon Software
